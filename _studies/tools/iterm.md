## 단축키 (Version 3)

### Split Panes
* 오른쪽에 생성 `Cmd + D`
* 아래쪽에 생성 `Cmd + Shift + D`
* 페인간 이동 `Cmd + Alt + Left/Right`

### Autocomplete
* 자동완성 `Cmd + ;`
* 명령 기록 `Cmd + Shift + ;`
* 복사 기록 `Cmd + Shift + H`
* 최근 경로 `Cmd + Alt + /`

### Instant Replay
* 다시보기 `Cmd + Alt + B`
